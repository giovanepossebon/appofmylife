## Specifications

- Swift 3
- iOS 10.3+

### Implemented Features:

- [x] Login
- [x] Home (Next episodes from my series)
- [x] TV show percentage completion
- [x] List of my series
- [x] Episode informations
- [x] Mark episode as watched
- [x] Profile
- [x] Search
- [ ] Add TV show to Watchlist
- [ ] Offline data
